== Installation ==

1. Upload `Simple Contact From` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use [contact] shortcode to add contact form in post and pages.
4. Use [contact email_to="Admin Email ID"] to change email from admin
